StyleTester is a Kirby CMS theme for creating Style Guides. You can use the StyleTester back-end to create, organize and comment html elements that will be rendered using your CSS and JS files. The StyleTester front-end will present these elements with comments, code snippets and all the controls you need to present them.

++++

You can use the same StyleTester install to house various Style Guides and if you wish share CSS and JS files among them.
